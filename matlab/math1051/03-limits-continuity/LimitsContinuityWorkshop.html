1.3 One-sided limits

We can also calculate the left-handed and right-handed limits. These are computed in MATLAB by passing the character strings 'left' and 'right' to the limit command as the last argument. For example:
syms x
g0 = (2 * x - 3)/(x - 1)
limit(g0, x, 1,'left')
limit(g0, x, 1,'right')
Remark: In this case we cannot omit the variable "x" in the command limit. Try it yourself! Delete "x" in line 15 or 16 from the previous code and re-run this section. You will get an error because MATLAB always requires the symbolic variable to calculate one-sided limits.
